# construct all combinations of input vector

pastes together all combinations of input vector

## Usage

``` r
get_all_combs(vec)
```

## Arguments

- vec:

  vector of strings

## Value

A vector of strings is returned.

## Examples

``` r
get_all_combs(letters[1:3])
#> [1] "b_a" "c_a" "a_b" "c_b" "a_c" "b_c"
```
